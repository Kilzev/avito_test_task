Блок 1. Создание объявления
## ТК-01 Успешное создание объявления

**Предусловия:**
- sellerId уникальный (111111–999999)

**Шаги:**
1. Отправить POST `/api/1/item` с валидным JSON
2. Передать sellerId, name, price, statistics

**ОР:**
- HTTP код 200
- В ответе присутствует поле:
```json
{
    "status": "Сохранили объявление - (UUID объявления)"
}
```

## ТК-10 Получение существующего объявления

**Предусловие:**
- объявление создано

**Шаги:**
1. Отправить GET `/api/1/item/{UUID объявления}`

**ОР:**
- HTTP код 200
- В ответе массив объектов
- Объект с нужным id присутствует
- Корректная структура:
```json
[
    {
        "createdAt": "<дата и время создания в формате ISO 8601 с часовым поясом>",
        "id": "<UUID объявления>",
        "name": "<название>",
        "price": "<цена>",
        "sellerId": "<id продавца>",
        "statistics": {
            "contacts": "<количество контактов>",
            "likes": "<количество лайков>",
            "viewCount": "<количество просмотров>"
        }
    }
]
```

## ТК-11 Получение несуществующего id

**Предусловие:**
- объявления с таким id нет

**Шаги:**
1. Отправить GET `/api/1/item/cbe09960-45a5-49d9-a6ae-dc0734cb7792` (или другой несуществующий UUID)

**ОР:**
- HTTP код 404
- Сообщение об ошибке в теле ответа:
```json
{
    "result": {
        "message": "item cbe09960-45a5-49d9-a6ae-dc0734cb7792 not found",
        "messages": null
    },
    "status": "404"
}
```
